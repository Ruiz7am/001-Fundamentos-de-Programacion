### Negación
Es el inverso del valor que tiene un objeto y tiene que ser distinto para ser true

### Conjunción

1 y 1 = 1
Los dos valores tienen que ser verdaderos para ser true

### Disyunción
1 y 0 = 1   
0 y 0 = 0
